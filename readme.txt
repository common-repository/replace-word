== Description ==

This plug-in allows you replace any word from site without edit code and without having any knowledge of development or HTML.
Also you can replace word with anchor tag with custom link added from admin settings.

We have added new feature same as anchor tag , now you can add heading tag also to the word using replace from admin side.

Here is some example:

If you want to replace word "Hi" with "Hello". just add both word in place of Find box and Replace Box.

== Installation ==

* Upload the replace word folder to the '/wp-content/plugins/' directory
* Activate the plugin through the 'Plugins' menu in WordPress
* The replace word option are in the settings sidebar menu.
* Click on the Add button on the replace word admin page to add as many replaces as you want.

== Frequently Asked Questions ==

= It's Store data of Replacement ? = 

Yes, It's store which word replace with which one and you can see it in admin.

= Can you replace multiple word? =

Yes, You can replace multiple word from site. You can also add Ancher tag with custom link.
